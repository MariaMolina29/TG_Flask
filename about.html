Inicio

¿Qué es AudioLab?

AudioLab es una herramienta interactiva diseñada para facilitar el análisis acústico de señales de audio de manera sencilla y visual. La plataforma permite a los usuarios capturar y analizar sonidos en tiempo real o cargar archivos .wav para un análisis más detallado.

¿Cómo funciona AudioLab?

La plataforma utiliza tecnologías avanzadas de procesamiento de audio, como la Web Audio API y WebSockets, para capturar, procesar y transmitir señales acústicas en tiempo real. La interfaz web se desarrolla en Python con Flask y Plotly, permitiendo una visualización interactiva de gráficos de oscilogramas, espectrogramas, gráficos de intensidad, y más.

¿Por qué usar AudioLab?

Ya sea para la investigación académica, el desarrollo de aplicaciones de voz, o simplemente para aprender más sobre el análisis de señales de audio, AudioLab ofrece una solución práctica y educativa. La herramienta permite experimentar con las propiedades del sonido en un entorno seguro y fácil de usar.

¿Qué tecnologías utiliza AudioLab?

  • Python y Flask: Para la estructura del backend y la transmisión de datos.
  • JavaScript y Web Audio API: Para la captura y manipulación de señales de audio en tiempo real.
  • Plotly.js: Para la representación gráfica interactiva de oscilogramas, espectrogramas y más.
  • Parselmouth: Para el análisis de características acústicas como pitch y formantes.
  • HTML, CSS y AOS: Para la creación de una interfaz amigable, responsiva y atractiva.
  • WebSockets: Para la comunicación bidireccional entre el navegador y el servidor.

Explora y aprende con AudioLab

AudioLab está diseñado para todas las personas interesadas en el sonido, desde estudiantes y académicos hasta músicos y desarrolladores. Úsalo para tus proyectos de investigación, para experimentar con sonidos naturales, o para analizar la voz humana de una manera completamente nueva.
